# 思政刷题系统

> 《习近平新时代中国特色社会主义思想概论》期末冲刺 · 纯前端刷题网页

一个无需服务器、打开即用的刷题网页应用。基于课本原文出题，支持分章节练习、模拟考试、智能错题本和数据统计，帮助高效复习期末考试。

**🎯 在线使用：**[k-ristina.github.io/si-xi-quiz](https://k-ristina.github.io/si-xi-quiz/)

---

## 功能模块

### 1️⃣ 分章节测试

按教材目录选择章节，逐题顺序练习。适合第一轮巩固复习。

- 动态生成章节卡片，显示做题进度
- 单选题 / 多选题 / 判断题：提交后即时判定对错并显示课本原文解析
- 浏览器本地记忆做题进度，下次可继续或重头开始

### 2️⃣ 模拟考试

打破章节限制，从全库随机抽题组卷，限时答题，模拟真实考场。

- 组卷规则：100 题（单选 40 + 多选 20 + 判断 40），Fisher-Yates 洗牌
- 答题中不提示对错、不显示解析，切题自由跳转
- 40 分钟倒计时，归零自动交卷
- 交卷后显示得分、耗时、正确率，可逐题查看错题解析

### 3️⃣ 智能错题本

错题自动收集，答对自动消除。

- 章节练习或模拟考试中做错的题自动存入本地
- 答对即消错，答错继续保留高频复练

### 4️⃣ 数据统计看板

量化展示复习进度，识别薄弱章节。

- 总复习进度、全库正确率、错题存量
- 各章节熟练度排行

## 自行部署

本项目是一个完全自包含的 HTML 文件，部署到任意静态托管平台即可。

### GitHub Pages

1. Fork 本仓库
2. Settings → Pages → Source 选 `master` 分支 → Save
3. 访问 `https://你的用户名.github.io/si-xi-quiz/`

### 其他平台

将 `index.html` 上传到任意静态文件服务（Vercel、Netlify、腾讯云 COS 等）即可。

---

## 本地使用

直接下载 `index.html`，用浏览器打开即可使用，无需安装任何依赖。

> ⚠️ 做题记录和错题数据存储在浏览器的 `localStorage` 中，清除浏览器数据或更换浏览器后记录会丢失。
